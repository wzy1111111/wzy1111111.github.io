# GitHub Pages 个人主页模板

## 使用方法

1. 在 GitHub 创建仓库：

```text
你的GitHub用户名.github.io
```

例如：

```text
wangziye.github.io
```

2. 把本文件夹里的内容上传到仓库根目录。

3. 进入仓库：

```text
Settings → Pages → Deploy from branch → main → /root → Save
```

4. 访问：

```text
https://你的GitHub用户名.github.io
```

## 需要替换的内容

- `Your Name`
- `中文名`
- 学校、实验室、论文标题、项目经历
- `assets/avatar.jpg`
- `assets/paper1.jpg`
- `assets/logo1.png`

如果没有图片，可以先删除对应的 `<img>` 标签。
